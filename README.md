
# Full Stack Employee Management Application Using ReactJS and Spring Boot


## Tech We Used

- ReactJs
- Spring Boot
- MySql Database
- Spring Security
- REST API
- JWT Auth
- Routing
- CORS

## Features

- Login
- Logout
- CURD Operations
- Data are stored in MySql database
- Neat and clean UI

## Steps to run in your machine

#### Run the following commands
```

Clone the rep

cd frontend
npm i
npm run start

cd backend
Run as Spring Boot
```
